# Detroit EMS Fleet Command

Real-time dispatch and resource tracking interface for Detroit Fire Department EMS.

## Features
- **35 EMS units** mapped across Detroit (32 DFD Medic + 3 HART)
- **12 receiving hospitals** with specialty capabilities
- **Tap-to-find** nearest available units and hospitals
- **Address search** with autocomplete
- **Filter by** battalion, availability, or specialty

## Live Demo
Open `index.html` in any browser or deploy to GitHub Pages.

## Usage
1. **Search**: Type address, tap Find
2. **Tap map**: Pin any location to see nearest resources
3. **Bottom sheet**: Swipe up for full unit/hospital list
4. **Filters**: Narrow by battalion or hospital capability

## Data
Currently uses simulated fleet positions. Ready for CAD integration.
